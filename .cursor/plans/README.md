# .cursor/plans/

Cursor IDE working plans / scratch space for in-progress workstreams. Mirrors the pattern from the sister `renewablesinfo_org` repo.

Plans here are typically transient — they capture current-thread planning, not committed roadmap. For substantive plans that should survive sessions, prefer:

- `docs/plans/` — the consolidation plan + per-notebook plans (the real planning artifacts)
- `docs/methodology/gaps_and_priorities.md` — the ranked v2 priority list
- `docs/decisions/` — ADRs for decisions worth permanently recording

Plans in this folder are fair game for cleanup once the workstream they describe is complete.
